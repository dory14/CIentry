untuk mencobanya di localhost, pastikan anda sdah menginstall apache,php, mysql

berikut ini cara menginstall script 
1. buat database dengan nama entry;
2. Import file entry.sql ke database tersebut
3. letakan file hasil extract di folder htdocs
5. pastikan server apache dan mysql jalan
6. lalu buka http://localhost/CIentry/
7.silahkan login dengan memasukan password dan username
  a).untuk masuk ke menu admin
	masukan username='admin' password='admin'
  b).untuk masuk ke menu superadmin
	masukan username='dori achmad' password='1'
